# Recitations

Mirror of QUL recitation segment JSONs used by the Mushaf mobile app.

Each subfolder is one reciter. Audio MP3 files are NOT hosted here —
the JSONs contain `audio_url` fields pointing to existing public CDNs
(`audio-cdn.tarteel.ai` for ayah-by-ayah, `download.quranicaudio.com`
for surah-by-surah).

Reciter slugs:
- `alafasy` — Mishary Rashid Al-Afasy (ayah, word-level segments)
- `husary` — Mahmoud Khalil Al-Husary (ayah, word-level segments)
- `minshawi` — Muhammad Siddiq Al-Minshawi (ayah, word-level segments)
- `al-shatri` — Abu Bakr Al-Shatri (ayah, word-level segments)
- `al-dosari` — Yasser Al-Dosari (ayah, word-level segments)
- `bandar-baleela` — Bandar Baleela (surah, ayah-level only)
- `muhammad-jibreel` — Muhammad Jibreel (surah, ayah-level only)
- `al-dussary-surah` — Yasser Ad-Dussary (surah, ayah-level only)
- `minshawi-children` — Minshawi With Children (surah, ayah-level only)

See `manaf/Recitation-Index` for the master manifest.
